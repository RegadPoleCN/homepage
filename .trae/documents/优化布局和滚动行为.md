## 需求分析

1. **浅色模式保留卡片样式** - 在浅色主题下，中栏的 profile-card 和 description-card 保留白色背景和阴影
2. **缩减中栏上部垂直比例** - 减小头像、间距等尺寸，让 footer 能在一屏内显示
3. **滚动行为优化** - 左栏和中栏固定，只有右栏可以上下滚动；左栏友链超出时可单独滚动
4. **手机端布局调整** - 左栏位于右栏下方（顺序：中栏 → 右栏 → 左栏）

## 技术实现

### 1. App.vue 布局修改
- 左栏和中栏设置 `position: sticky` 或 `fixed`，`height: 100vh`，`overflow-y: auto`
- 右栏保持正常滚动
- 手机端使用 `order` 属性调整顺序：中栏 `order: 1`，右栏 `order: 2`，左栏 `order: 3`

### 2. CenterPanel.vue 样式修改
- 缩小头像尺寸（120px → 80px）
- 减小内边距和间距
- 添加 `[data-theme="light"]` 选择器，在浅色模式下应用卡片样式（白色背景 + 阴影）

### 3. LeftSidebar.vue 样式修改
- 添加 `overflow-y: auto` 支持独立滚动
- 添加 `[data-theme="light"]` 选择器，在浅色模式下应用卡片样式

### 4. 响应式布局
- 手机端（≤1024px）：中栏在最上面，右栏在中间，左栏在最下面，footer 在最底部

## 修改文件
