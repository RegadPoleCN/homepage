## 增强浅色和暖阳主题的卡片样式

当前问题：卡片阴影和边框效果不够明显，与背景融为一体。

### 解决方案
通过以下方式增强卡片的立体感和视觉层次：

1. **更深的阴影**
   - 多层阴影效果：`box-shadow: 0 4px 6px rgba(0,0,0,0.07), 0 12px 28px rgba(0,0,0,0.12), 0 20px 40px rgba(0,0,0,0.08)`
   
2. **更明显的边框**
   - 边框颜色加深：`border: 1px solid rgba(0, 0, 0, 0.12)`

3. **调整浅色主题的背景色差异**
   - 修改 `site.config.ts` 中浅色和暖阳主题的 `centerBg` 颜色，让它与卡片背景有更大差异
   - 浅色：`centerBg` 从 `#ffffff` 改为 `#f0f4f8`
   - 暖阳：`centerBg` 从 `#ffffff` 改为 `#fef9f0`

### 修改文件
- `src/components/CenterPanel.vue` - 增强卡片阴影和边框
- `src/components/LeftSidebar.vue` - 增强卡片阴影和边框
- `src/config/site.config.ts` - 调整背景色差异
