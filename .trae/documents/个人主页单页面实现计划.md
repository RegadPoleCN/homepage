## 项目概述

基于现有的 Vue 3 + Vite + TypeScript 项目，创建一个三栏布局的个人主页单页面应用。

## 页面布局设计

### 三栏结构

* **左栏（最小）**：友链区域，可通过配置文件配置

* **中栏（突出显示）**：个人简介，使用阴影/颜色突出层次感

* **右栏（最大）**：主要内容展示区域

* **底部**：备案号、公安备案、版权信息（可配置）

### 视觉效果

* 使用 CSS 阴影和颜色实现分层效果

* 中间栏通过 `box-shadow` 和 `z-index` 突出显示

## 主题切换功能

### 外部依赖

* **@vueuse/core**：使用 `useDark` 和 `useColorMode` 实现主题切换

* **@iconify/vue**：图标库，用于美化界面和设置按钮

### 设置菜单

* 位于屏幕正中央的圆角长方形弹窗

* 背景使用毛玻璃效果（`backdrop-filter: blur()`）

* 预设多种色调主题供选择

### 预设色调

* 浅色主题（Light）

* 深色主题（Dark）

* 蓝色主题（Ocean）

* 绿色主题（Forest）

* 紫色主题（Purple）

* 暖色主题（Warm）

## 技术实现

### 文件结构

```
src/
├── App.vue                    # 主应用组件
├── config/
│   └── site.config.ts         # 站点配置文件
├── components/
│   ├── LeftSidebar.vue        # 左栏 - 友链
│   ├── CenterPanel.vue        # 中栏 - 个人简介
│   ├── RightPanel.vue         # 右栏 - 主内容
│   ├── Footer.vue             # 底部 - 备案信息
│   └── SettingsModal.vue      # 设置弹窗
├── composables/
│   └── useTheme.ts            # 主题切换逻辑（基于 @vueuse/core）
└── style.css                  # 全局样式
```

### 配置文件内容

* 友链列表（名称、链接、图标）

* 个人简介信息

* ICP 备案号

* 公安备案号

* 版权信息

* 预设主题配置

## 实现步骤

1. **安装依赖** - `pnpm add @vueuse/core @iconify/vue`
2. **创建配置文件** - `src/config/site.config.ts`
3. **创建主题管理** - `src/composables/useTheme.ts`（使用 VueUse）
4. **创建左栏组件** - 友链展示
5. **创建中栏组件** - 个人简介
6. **创建右栏组件** - 主内容区
7. **创建底部组件** - 备案信息
8. **创建设置弹窗** - 主题切换（使用 Iconify 图标）
9. **重构 App.vue** - 整合所有组件
10. **更新全局样式** - CSS 变量和主题
11. **测试验证** - 启动开发服务器验证效果
